Aufgabe 1 - Bilderkennung mit ml5

Herangehensweise

Die Umsetzung der ersten Aufgabe erfolgte in mehreren Schritten. Zunächst wurden die Kapitel ELT und LFD des Lernmaterials durchgearbeitet und mit den Tutorials und Dokumentationen Wissen zum ml5-Framework aufgebaut. Im nächsten Schritt ist das Angular Projekt aufgesetzt und die Bibliothek zur Klassifizierung von Bildern integriert worden. Danach wurden weitere Kriterien wie Visualisierung der Ergebnisse mittels eines Balkendiagrammes und das Hochladen von eigenen Dateien (im richtigen Dateiformat) umgesetzt. Im letzten Schritt kam Bootstrap und CSS fürs Erstellen und Stylen des Layout zum Einsatz.

Quellen und Bibliotheken

  • Ml5 Tutorials / Dokumentation: Ml5.js Dokumentation und Ml5 Image Classification Tutorial
  • Angular (Frontend Framework)
  • Bootstrap (Frontend Framework)
  • Chart.js (Datenvisualisierung)
